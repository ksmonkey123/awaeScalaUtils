trait Ref[+A, T <: RefType[T]] extends () ⇒ Option[A]

Common base trait for all reference types. References are created by the companion object Ref. All references are read-only, however immutability of the referenced object can obviously not be ensured.

A

the data type of the referenced object (covariant)

T

  2. abstract def exists: Boolean

    Indicates if the referenced object exists.

    Indicates if the referenced object exists.

    For auto-recovering references this indicates the actual state of the reference. Therefore if this returns false the next method that can recreate references will recreate the reference.

    returns

    true if the referenced object exists, false otherwise.

    Note

    this method cannot be used for existence checks on the referenced object since any further operation is not atomic and therefore GC may clear the reference in the meantime. If existence of the object is relevant use the following implementations using apply:

    ref.apply().foreach(x => /* code here */)
ref().foreach(x => /* code here */)

  24. def stripped(update: Boolean = false): Ref[A, T]

    converts this into a reference that is stripped of all secondary data.

    converts this into a reference that is stripped of all secondary data.

    Here special rules apply to different reference types:

    • if this is a normal reference, this is directly returned
    • if this is an auto-recovering reference, the reference is recovered iff update==true and the object referenced by this reference does not exist. The referenced object is packed into a normal reference of the same type.
    update

    indicates whether or not the reference should be updated before stripping it of the secondary data.

    returns

    a new basic reference of the same type referencing the same object (if it exists) as this reference.

    Note

    forcing a reference update by setting update to true does not guarantee that the object referenced by the resulting reference does actually exist. It is entirely possible for GC to clear the newly generated object at any time - even before the reference is actually returned.
